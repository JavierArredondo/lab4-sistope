Instrucciones de uso:

1. Ejecutar el make en la carpeta principal.
2. Entrar a la carpeta bin.
3. Ejecutar programa de la siguiente forma :

	* ./classifier.out -c 20 -u 50 -n 50 -b 1
	* ./classifier.out -c 20 -u 50 -n 50 -b 0

 	- Donde son los parametros que se encargan de clasificar a la imagen y tambien ingresar la bandera si desea que se muestre por pantalla el resultado obtenido.

4. Las imagenes generadas se encuentras en la carpeta imagesbin.



***** IMPORTANTE : Las imagenes no se encuentran cargadas y deben colocarse en la carpeta de images: con la siguiente nomenclatura:
		- image_1.bmp
		- image_2.bmp
		- image_3.bmp
		- image_4.bmp
		- etc.
